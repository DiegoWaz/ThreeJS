# Three JS - Minecraft

### Installation

npm i

### Run project
npm run dev

### Commment jouer

Deplacer vous aves les touches directionnels

Creer des cubes :

- touche A -> cube gris
- touches Z -> cube en bois
- touche E -> cube rose

Supprimmer un cube :

- touche S

Changer la couleur :

- touche Y U I
